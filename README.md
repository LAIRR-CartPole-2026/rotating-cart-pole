# Rotating Double Inverted Cart-Pole System
### LAIRR Summer Project 2026

## Overview
This repository contains all design, hardware, firmware, and software files
for the Rotating Double Inverted Cart-Pole System developed as part of the
LAIRR Summer Project 2026. The system is a research and teaching platform
for control theory, reinforcement learning, and robotics.

## Repository Structure
| Folder | Description |
|---|---|
| /mechanical | CAD models, drawings, and fabrication documentation |
| /electronics | Schematics, wiring diagrams, and component lists |
| /embedded | Microcontroller firmware and low-level software |
| /control | Mathematical models, simulations, and controller designs |
| /software | ROS2 nodes, Python scripts, and high-level experiment tools |
| /docs | Meeting notes, SRD, reports, and supervisor materials |
